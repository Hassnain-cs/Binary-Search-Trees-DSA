# Binary Search Tree (BST) in C

## Project Overview
This project implements a Binary Search Tree (BST) in C. The program inserts a random number of lowercase characters (between 11 and 20) into the tree and performs operations such as traversal, counting nodes, and calculating tree height.

---

## Features
- Insert characters into a Binary Search Tree  
- Search for a character  
- Count total nodes  
- Determine tree height  
- Display sorted output using inorder traversal  
- Free dynamically allocated memory
